第三章学习小结

列表即是一列按照特定的顺序排列的元素。列表元素的索引从左往右由0开始升序排列，从右往左由-1开始降序计数，使用方法：list = ["z","y","c"]	list[0]=>"z", list[-1]=>"c",list[1] = list[-2]=>"y".

在使用list的同时我们可以使用字符串函数。例如：print(bicycles[0].title())

列表是动态的，可以通过列表函数进行增删改操作。

增：.append()可以再list的末尾加上一个元素
      .insert(x,'A')可以在指定的位置加上一个元素

删：del list[x] 直接删除某一元素
       list_poped = list.pop(x) 删除某一元素，并且可以使用 若无参数，即弹出末尾元素。
       list.remove('A') 可以根据值来删除元素，但是只会删除第一个这个元素，若该元素值在列表中有重复需要使用循环结构。

改：list[x] = 'A' ,即可修改。

排序方法：
	.sort(reverse = True)字母逆序排列，无参数即可正向排列
	sorted(list,reverse = True)临时字母逆序排列，不会改变真正的list元素排列顺序，无后面参数即可正向排列
	list.reverse()可以将list元素逆序排列
len(list) 可以确定列表长度
