#检查相等
#用|=| 来比较两个值是否相等，若相等返回True,否则返回False.
# 在Python中检查相等时是区分大小写的，若要实现不区分大小写，可以将要识别的变量值转化为小写，在做比较
#car.lower() = 'audi'

#检查不相等
#用|！=| 来比较两个值是否不相等，若不相等返回True,否则返回False.

#检查多个条件时的时候，将需要同时满足的条件用and连接，将只需要满足其中之一的多个条件用or连接。

#检查特定值是否在列表中时 用in 检查是否不在列表中时，使用not in
例：
if user not in banned_users:
    print(user.title()+", you can post a response if you wish.")

#布尔表达式
即是条件测试的一个别名，一个变量的定义名称，例如：can_edit = False.

if-elif-else条件格式
if (条件一)：
    （满足条件一执行语句）
elif (条件二)：
    （满足条件二执行语句）
elif (条件三)：
    （满足条件三执行语句）
else:
      (以上条件都不满足执行语句)

#使用if语句来处理列表
常用的步骤就是先将列表遍历，然后在遍历中添加if条件语句。
也可以用来确保列表不为空，例如：
list = []
if list:
    (语句块一)
else:
    (语句块二)

当列表为空的情况下，执行语句二。
