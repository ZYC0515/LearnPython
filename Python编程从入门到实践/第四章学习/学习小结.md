第四章学习小结

本章中，我们学习看用for循环来遍历整个列表的操作：
for xx in xxs:
    print(xx)

在使用for循环结构的时候需要注意的是|：|在写完一个for语句后 千万不能忘记它的冒号，那么在写for循环结构中的执行语句是需要注意缩进，Python是一款严格控制缩进的语言。

在学习完for循环结构后，我们学会了使用range()函数来生成一系列的数值：
for value in range(1,6):#这里生成的是1-5
    print(value)
>>>[1,2,3,4,5]

通过list()函数能够将range()结果直接转换为列表：
numbers = list(range(1,11,2))#最后一个参数是步长数值，可以通过这个生成偶数奇数等等等等。
    print(numbers)
>>>[1,3,5,7,9,11]

这里介绍一种更加方便生成所数值列表的方法：列表解析法：
squares = [value**2 for value in range(1,11)]
print(squares)
>>>[1,4,9,16,25,36,49,64,81,100]

通过min(squares)`max(squares)`sum(squares)函数可以分别求最小值、
最大值以及和。

学习完上一部分知识后，我们学习了使用切片，来对列表中的部分数据进行提取。
num = ['a','b','c','d','e','f']
print(num[1:4])
>>>['b','c','d']

num[a:b]
切片就是从列表中序号为a的元素开始一个个提取出来，直到识别到序号为b的序号停止遍历，因此序号为b的元素没有被提取出来，被提取出来的是从序号a到序号b前一位的所有元素。

复制列表也是一个很有实用价值的功能。b = a[:] a列表中的元素就复制给了b，并且a、b两个列表相互独立各不干涉，若直接将b = a,两个表就会关联成一个表，改变一个，另一个也会改变。

元组的概念，其实就是不可改变的列表。若要改变元组中的数据，只能重新对元组进行赋值。
