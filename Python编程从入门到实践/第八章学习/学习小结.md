第八章学小结
首先学习了如何定义函数和使用函数。
#用def来定义函数，调用函数的时候直接用函数名。
#分清实参和形参的区别，实参就是真实地传递给函数的值，形参就是相当于函数中的一个占位符。
#给函数传入参数的时候可以给定形参名再加参数，这样就不用管顺序了。
#函数是可以多次调用的。
#函数的参数可以设置默认值，有默认值的形参要放在无默认值的形参之后，所谓的设置默认值就是在函数中直接给某个形参赋值。
#return可以返回值给程序
#参数可以使列表、字典，只要在函数内部使用该参数的时候需要遵循该实参的使用方法和规则，例如向函数传递列表，在函数的内部使用该参数时就要使用列表的相关函数对该参数进行处理。
#明了的函数名能够增加代码的可读性
#传入列表参数时，使用列表副本就不会造成原始列表的改变。（还有疑问）
#当有未知数量的实参要被引入函数时，使用*【形参】，例如def make_pizza(*toppings):这样能够创建一个元组，将所有的实参都封装进去。
#需要传入不同类型的参数时，可以结合使用位置实参和任意数量实参。
#传入任意数量的关键字实参 用**【形参】，这里创建一个名为形参的空字典，并将所有收到的键值对存入这个字典中
#将函数存储在模块中，就是将函数存储在一个py文件中，使用时直接import
使用模块中的函数，需要加 |模块.模块中的函数| 来使用函数
#若要导入模块中的特定函数，使用from 模块 import 函数 来导入
#可以使用import 模块 as 模块别名 来给模块指定别名来方便程序的使用。
#若要导入模块中的所有函数 from 模块名 import * （不推荐使用，尤其是在使用别人的模块时，如果有重名会互相覆盖导致程序报错。）、


本章学习大约用时 2-2.5小时。
